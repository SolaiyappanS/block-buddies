# Block Buddies - Gameplay Guide

## Objective
Guide all players from their starting positions to the exit door in each of the 5 levels. Work together to solve puzzles using rocks and avoiding hazards.

## Controls
- **Arrow Keys** or **WASD** - Move your character
- **Mouse** - Click buttons in menus and alerts

## Game Mechanics

### Movement
- Use arrow keys (‚Üë‚Üì‚Üê‚Üí) or WASD to navigate the grid
- You cannot move through other players
- You cannot move through obstacles (river, lava, wind) - this restarts the level

### Rocks
- **Pushable objects** that help you navigate
- Approach a rock and press a direction to **push it**
- The rock moves in that direction (you can't push it off the grid)
- Rocks cannot be pushed into other rocks or the exit

### Obstacles
- **River** (blue water icon) - Impassable water
- **Lava** (red fire icon) - Burning hazard
- **Wind** (gray wind icon) - Blocked passage

### Rock-Obstacle Interaction
When you push a rock **onto** an obstacle:
1. The rock covers the obstacle (it becomes invisible)
2. The rock now occupies that space
3. The hazard is temporarily neutralized
4. When you move the rock away, the obstacle reappears

**Strategy**: Use rocks to cover hazards and create safe paths!

### Exit Door
- **Yellow door icon** at the end of each level
- **ALL players must reach it simultaneously**
- Once all players are at the exit, you advance to the next level

## Level Progression

### Level 1: Tutorial
- 3 starting positions for players
- Simple rock configuration
- Lava obstacles to avoid or cover
- Learn the basics!

### Level 2: Complexity
- 2 starting positions
- More rocks to manipulate
- Mixed river and lava obstacles
- Puzzle-solving required

### Level 3: Wind Challenge
- 3 starting positions
- Wind obstacles introduced
- Larger layout
- Increased difficulty

### Level 4: Maze Master
- 3 starting positions
- Complex rock arrangement
- Multiple hazard types
- Advanced puzzle-solving

### Level 5: Ultimate Challenge
- 2 starting positions
- Maximum complexity
- Intricate rock pathways
- All obstacle types present

## Winning
- Complete all 5 levels with all players
- Reach the exit on Level 5
- See the congratulations message!

## If You Hit an Obstacle
A popup alert will appear giving you two options:

1. **Restart Level** - Start the current level over
2. **Start From Level 1** - Begin the entire game from scratch

All player positions reset accordingly.

## Tips & Tricks

### Working as a Team
- Communicate with other players
- Plan rock movements before pushing them
- Make sure everyone can reach the exit

### Problem Solving
- Map out the obstacles
- Identify which rocks can help
- Plan a safe path to the exit
- Test rock positions before final movement

- ‚ùå Pushing rocks without a plan (they're hard to move back!)
- ‚ùå Forgetting that ALL players must reach the exit
- ‚ùå Not checking if the exit is reachable from all starting positions

## Game Creation

### Creating a New Game
1. Click **"Create Game"** from the main menu
2. You start at Level 1 automatically
3. A game code is generated
4. Share the code with other players

### Joining a Game
1. Click **"Join Existing Game"**
2. Enter the game code (shared by the creator)
3. You join at Level 1 with a unique starting position
4. The game begins immediately

## Game Code
The 8-character code at the top of the game screen identifies your game. Share this with friends to play together!
